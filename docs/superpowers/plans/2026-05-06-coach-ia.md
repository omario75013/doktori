# Doktori Phase 2 #9 — Coach IA Kimi implementation plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a feature-flagged-OFF Coach IA chat that, when activated, orients patients to specialties via a Kimi-powered conversation with hardcoded safety rails (disclaimer, red-flag redirect, rate limit, cost cap, no content logging).

**Architecture:** Server-rendered `/coach-ia` page → client chat UI → POST `/api/coach-ia` → auth + flag + rate limit + cost guard → OpenRouter Kimi streaming completion. Zero content logged; only metadata (tokens, cost, latency) in `coach_ia_usage` table. Mandatory disclaimer modal gates first interaction.

**Tech Stack:** Next.js 16 RSC + client islets · Drizzle ORM · ioredis · OpenRouter API (OpenAI-compatible) · vitest.

**Spec:** `docs/superpowers/specs/2026-05-06-coach-ia-design.md` (HEAD `a0b3a16`).

**Critical pre-implementation checklist (Omar)**:
- [ ] Physician sign-off on the system prompt
- [ ] Legal review of the disclaimer modal text
- [ ] DPIA V1 reviewed and accepted

Implementation can ship without these (feature flag OFF), but **flag flip in prod requires all three**.

```ts
export const coachIaUsage = pgTable(
  "coach_ia_usage",
  {
    id: uuid("id").primaryKey().defaultRandom(),
    patientId: uuid("patient_id").references(() => patients.id, { onDelete: "set null" }),
    eventType: varchar("event_type", { length: 40 }).notNull(),  // 'message' | 'disclaimer_accepted'
    tokensIn: integer("tokens_in").default(0),
    tokensOut: integer("tokens_out").default(0),
    costUsd: numeric("cost_usd", { precision: 10, scale: 6 }).default("0"),
    latencyMs: integer("latency_ms").default(0),
    error: text("error"),
    createdAt: timestamp("created_at", { withTimezone: true }).defaultNow().notNull(),
  },
  (table) => [
    index("coach_ia_usage_patient_idx").on(table.patientId),
    index("coach_ia_usage_created_idx").on(table.createdAt),
  ]
);
```

```ts
import RedisMock from "ioredis-mock";

const { buildSystemPrompt, checkRateLimit, estimateCost } = await import("./coach-ia");

describe("buildSystemPrompt", () => {
  it("includes the SAMU 190 redirect rule", () => {
    const prompt = buildSystemPrompt();
    expect(prompt).toContain("190");
  });
  it("includes specialties from shared constants", () => {
    const prompt = buildSystemPrompt();
    expect(prompt).toContain("cardiologue");
    expect(prompt).toContain("pediatre");
  });
  it("forbids diagnosis", () => {
    const prompt = buildSystemPrompt();
    expect(prompt.toLowerCase()).toContain("ne jamais poser de diagnostic");
  });
});

describe("checkRateLimit", () => {
  it("allows first 10 messages in 24h", async () => {
    for (let i = 0; i < 10; i++) {
      const allowed = await checkRateLimit("patient-A");
      expect(allowed).toBe(true);
    }
  });
  it("blocks 11th message in 24h", async () => {
    for (let i = 0; i < 10; i++) await checkRateLimit("patient-B");
    const allowed = await checkRateLimit("patient-B");
    expect(allowed).toBe(false);
  });
});

describe("estimateCost", () => {
  it("computes Kimi pricing correctly", () => {
    const cost = estimateCost(1000, 500);
    // 1000 in × $0.40/M + 500 out × $2.00/M = 0.0004 + 0.001 = 0.0014
    expect(cost).toBeCloseTo(0.0014, 6);
  });
});
```

- Create: `apps/web/app/api/coach-ia/route.ts`

- [ ] **Step 1: Implement POST handler** with:
  - `requireAuth(req)` → must be `role==="patient"`
  - `isEnabled("coach_ia_enabled")` → return 403 if false
  - `checkRateLimit(patientId)` → 429 if exceeded
  - `checkGlobalRateLimit()` → 429 if exceeded
  - `checkCostCap()` → 429 if cap reached
  - validate request body schema (Zod): `messages: [{role: 'user'|'assistant', content: string max 2000 chars}]`, last message must be from user
  - call `streamCompletion(messages)` → return as `ReadableStream` with `Content-Type: text/event-stream`
  - on stream close: insert `coach_ia_usage` row with tokens + cost + latency, NO content
